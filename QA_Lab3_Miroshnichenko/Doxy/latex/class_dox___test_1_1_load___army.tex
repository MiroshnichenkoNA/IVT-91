\doxysubsection{Подробное описание}
Класс для снаряжения армии. 

Описывает снаряжение войска, включает в себя один объект, реализующий интерфейс оружия, и один объект, реализующий инферфейс солдат. Также имеет поле рейса отправки. ~\newline
